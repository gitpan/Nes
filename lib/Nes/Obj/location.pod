
=head1 NAME

Nes::Obj::location - Redirect to page.

=head1 SYNOPSIS

    {: include ('{: * cfg_obj_top_dir :}/location.nhtml', 
                 URL,
                 [STATUS] 
               ) 
    :}

=head1 DESCRIPTION

Redirect to especific page.

=head1 PARAMETERS

=over 2

=item URL

URL to redirect.

=item STATUS

HTTP status, default is "302 Found"

=back

=head1 Samples

Redirect to index by "302 Found" HTTP status:

    {: include ('{: * cfg_obj_top_dir :}/location.nhtml', 
                './'
               ) 
    :}

Redirect to http//example.com/ by "301 Moved Permanently":

    {: include ('{: * cfg_obj_top_dir :}/go_language.nhtml', 
                'http//example.com/',
                '301 Moved Permanently'
               )
    :}
