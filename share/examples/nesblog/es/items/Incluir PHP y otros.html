Ejemplos de include

Descripción

Con include podemos incluir casi cualquier tipo de archivo; HTML, PHP, PERL, PYTHON, NSQL, NHTML, SH, TXT, etc.

Include PHP

Muchas veces tenemos que incluir en nuestras páginas partes de código de terceros; publicidad, estadísticas, etc. las más de las veces éstas vienen en PHP. Nes es compatible y permite incluir archivos PHP con sólo hacer esto: 

        {: include ('file.php') :}

El siguiente es un script en PHP que muestra un calendario:

{: include ../../scripts/calendar.php :}

Puedes incluir Tags Nes en cualquier parte del archivo PHP.

Sólo se soporta el método GET.



Include Perl

Cuando incluimos un script Perl con sentencias print, Nes redirige la salida. 

        {: include ('file.pl') :}
{: include ../../scripts/perl.pl :}



Include Python

Ejemplo de Python. 

        {: include ('file.py') :}
{: include ../../scripts/python.py :}



Include SH

También podemos incluir archivos de shell. El siguiente es un script en SH que muestra el comando cal de unix: 

        {: include ('file.sh') :}
        

{: include ../../scripts/cal.sh :}



Include nHTML

Cuando incluimos un nhtml de Nes también se ejecuta su script. 

        {: include ('file.nhtml') :}
{: include ../../scripts/suerte.html :}



Include HTML

Se puede incluir cualquier archivo HTML normal tipo text/html. 

        {: include ('file.html') :}
{: include ../../scripts/solo_html.html :}



Include TXT

Cualquier archivo de texto plano (text/plain). 

        {: include ('file.txt') :}
{: include ../../scripts/archivo.txt :}


(Utiliza "Ver fuente" del menú superior para ver más detalles)
