NES 0.9 pre

Qué puede hacer por ti NES

Con NES puedes hacer lo mismo o más que con cualquier otro sistema de plantillas solo que mucho más rápido y mucho más fácil. El tiempo es el recurso más valioso.

0.9 pre

La 0.9 es una versión pre-release de la 1.0. A partir de la cual se garantiza la compatibilidad entre versiones.

Esta versión no es compatible con las anteriores. Ha cambiado el prefijo del nombre de las variables de entorno, env_* son las del entorno CGI y nes_* las del entorno NES, antes todas eran env_*.

Simplicidad

NES dispone de únicamente diez Tags para incluir en el HTML, y un sólo objeto con un único método que instanciar en Perl. Podrás empezar a trabajar con NES en horas, y no en semanas o meses como en otros sitemas.

Seguridad

Gestión transparente de inyección de código y cross-site scripting. Plugins para crear formularios seguros; Forms y Captcha.

Objetos

Paradigma orientado a objetos que te permitirá reutilizar tu código o usar librerías de terceros. En cada versión de Nes se irán añadiendo nuevos Objetos.

Wrapper

Soporte total de PHP Wrapper. Puedes utilizar NES como Wrapper para añadir funcionalidad a otros tipos como HTML, o en PHP para intercambiar información. En la Web de ejemplo lo hacemos con HTML, en los que ahora podemos incluir contenido dinámico.

mod_perl

Compatibilidad con mod_perl.

Sesiones

Creación y gestión de sesiones de usuario.

Plugins

Se le puede añadir funcionalidad a NES mediante Plugins que puedes crear tu mismo o usar los de terceros.

PHP y otros

Para insertar partes de código PHP en Nes sólo hay que teclear un Tag, Nes permite la inserción en tus plantillas de script en otros lenguajes distintos a Perl sin escribir una sola linea de código Perl. Además la mayoría de ellos pueden contener Tags Nes y se tratados como plantillas de Nes.

SQL

Puedes insertar sentencias SQL SELECT en tus plantillas directamente.

Archivos de configuración

NES provee de archivos de configuración ( .nes.cfg ) al estilo de los .htaccess que podrás utilizar también para tus script.
