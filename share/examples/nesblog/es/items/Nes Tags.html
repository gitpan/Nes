Resumen Tags de Nes

Índice Tags

{: NES ver (file) :} Se coloca principio del archivo nhtml y "file" es el script a ejecutar
{: # comment :} Comentario, se elimina
{: $ variable :} Se sustituye por el valor de "variable"
{: * environment_variable :} Se sustituye por el valor de "environment_variable"
{: ~ (expresión) HTML :} Se muestra el bloque HTML si expresión resulta cierto
{: sql (SQL SELECT) HTML :} Se repite el bloque HTML tantas veces como registros tenga "table"
{: @ (table) HTML :} Se repite el bloque HTML tantas veces como registros tenga "table"
{: @$ table.field :} Se sustituye por el campo "field" de "table"
{: include (file) :} Se sustituye por el archivo "file", file puede ser un nhtml, html, php, sh, text, etc.
{: & tag (param...) :} Resevado para el uso por plugins
